## 📐 Теория: Метод двух указателей

**Метод двух указателей** — алгоритмическая техника для работы с подстроками. Два указателя `l` (левый) и `r` (правый) задают границы скользящего окна по строке. Оба указателя движутся **только вправо**, что даёт сложность **O(n)**.
